Platform is a cross-platform rendering library and API.


 Supported Platforms
 -------------------

 * Windows 10
 * Linux
 * PS4
 * Xbox One

Supported Rendering API's
-------------------

 * Direct3D 11
 * Direct3D 12
 * PS4
 * OpenGL
 * Vulkan

 Required
 ---------

 * CMake
 * Python 3
 * Git
 * GitPython
 * Visual Studio 2017 or later (Windows)
 * Clang 8+ (Linux)
